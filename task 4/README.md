Task 4: Usability Heuristic Evaluation

Overview

This project analyzes a poorly designed local business website using Jakob Nielsen's 10 Usability Heuristics. The goal is to identify usability issues and propose wireframe-based solutions to improve the user experience.

Key Findings

- Cluttered homepage
- Confusing navigation
- Inconsistent design
- Poor mobile responsiveness
- Limited user feedback

Proposed Solutions

- Simplified navigation
- Improved content hierarchy
- Consistent UI design
- Better error handling
- Mobile-friendly layout

Tools Used
- Google Slides
  
Outcome
The redesigned wireframes improve usability, accessibility, and overall user satisfaction, helping create a more effective and user-friendly website.
