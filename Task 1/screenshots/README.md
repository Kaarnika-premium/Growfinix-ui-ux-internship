# Task 1: User Persona & Journey Mapping (Luxury Event Booking)
## Overview
This project focuses on researching and designing the user experience for a luxury event booking service. The goal is to understand user needs, create user personas, map their journey, and identify pain points and opportunities throughout the booking process.
## Objective
- Research users of a luxury event planning service.
- Create two distinct user personas.
- Map the complete user journey.
- Identify pain points.
- Discover opportunities for improvement.
## Tools Used
- FigJam
## User Persona 1: Ray (Customer)
### About
Ray is a customer looking for premium event planning services for special occasions.
### Goals
- Find reliable event planners.
- Get personalized recommendations.
- Understand pricing clearly.
- Enjoy a smooth booking experience.
### Frustrations
- Lack of trust in planners.
- Hidden costs.
- Poor communication.
- Delayed responses
## User Persona 2: Richard (Event Manager)
### About
Richard is an event manager responsible for handling customer requirements and delivering premium event experiences.
### Goals
- Understand client needs.
- Create customized event plans.
- Maintain clear communication.
- Deliver successful events.
### Challenges
- Managing multiple clients.
- Handling last-minute changes.
- Communication gaps.
- Coordinating vendors.
## User Journey
### Customer Journey
1. Discovers the service.
2. Explores available event packages.
3. Shares ideas and requirements.
4. Receives recommendations.
5. Chooses services and package.
6. Confirms booking.
7. Receives event updates.
8. Experiences the final event.
### Event Manager Journey
1. Receives customer inquiry.
2. Understands requirements.
3. Creates event plan.
4. Coordinates resources.
5. Communicates updates.
6. Executes the event.
7. Delivers premium experience.
## Pain Points
- Trust issues between customers and planners.
- Lack of transparent pricing.
- Communication gaps.
- Last-minute changes.
- Difficulty comparing services.
## Opportunities
- Personalized recommendations.
- Verified planner profiles.
- Transparent pricing.
- Real-time communication.
- Planning dashboard.
- Digital portfolios and previews.
## Design Process
- User Research
- Persona Creation
- Journey Mapping
- Pain Point Analysis
- Opportunity Identification
- Flowchart Design
